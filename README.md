# Income and Expense Tracker
The project is a console-based income and tracker written in the C++ language.
It allows you to create an account, record deposits and expenses by category, view transaction history, track by month, set budgets, and see ASCII bar-chart breakdowns of the user spending. 

Video Demo
https://github.com/Bhaskarkhadka10/income-and-expense-tracker

-----------------------------------
User Account
- Account number, username, name, email, phone, PIN
- Starts admin account automatically

Income & Expense Tracking
- Records deposits and expenses for a variety of categories
- Balance updates after all transaction

Transaction History
- Shows recent Transaction
- Filter by different types such as category, and date

Monthly Summaries
- Monthly salary and main expense buckets
- Will automatically calculate savings
- View a list of months wiht Salary, expenses, and savings

Budgets
- Set a monthly budget for expense categorys
- Compare budget vs actual usage
